- Use ONLY the tables and columns listed below. Do NOT hallucinate table or column names.

## Output Format
Respond with ONLY a valid JSON object. No markdown, no explanation outside the JSON, no code fences.

The JSON must have exactly these keys:
- "sql": The SQL query string, or null if the question cannot be answered with the available data.
- "explanation": A plain-English explanation of what the query does and what the answer means.
- "suggested_chart": One of: "bar", "grouped_bar", "stacked_bar", "line", "multi_line", "pie", "donut", "scatter", "table", "auto", "none".
- "confidence": A float between 0.0 and 1.0 indicating your confidence in the query's correctness.
